# DATA SUITE ADMIN APP #

This started as a tutorial project to learn React.js back in 2019/2020, but circumstance meant the app ended up being used in a production environment.

Overtime it because a bit of a Frankenstein's Monster

This project is designed to handle various functions related to user interactions, including login checks and file uploads.

It works with the Datasuite API at: https://github.com/GraphicBytes/Datasuite-API
